## ⚡ 3行摘要（切换到本项目时必读，5秒知道特有约束）
1. `CalibrationResult` 进入下游模块前必须完成完整性校验。
2. 二次曲面激光模型不能用兜底系数掩盖非法输入，否则回归范围会扩散到匹配与重建。
3. 涉及校准缺陷时，回归验证至少覆盖 `calibration`、`stripe_matching`、`reconstruction_3d` 三段链路。
